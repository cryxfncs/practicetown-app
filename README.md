> The official, blazingly fast desktop client for the Practicetown ecosystem.

Practicetown Workspace is a native desktop application designed to manage premium Pterodactyl blueprints, themes, and extensions. By leveraging the power of Rust and Tauri, this app provides a lightweight, highly optimized alternative to traditional web dashboards.

## ✨ Features

*   **Native Performance:** Minimal RAM and CPU footprint compared to Electron apps.
*   **Developer-First UX:** Clean, distraction-free interface inspired by the GitHub dark theme.
*   **Integrated Ecosystem:** Direct, secure access to the entire Practicetown network (WatchPaste, GPT, Portfolios).
*   **Secure by Design:** Built-in session locking mechanism.

---

## 🛠️ Prerequisites

Before you clone the repository, make sure you have the following installed on your machine:

1.  [Node.js](https://nodejs.org/) (LTS recommended)
2.  [Rust](https://rustup.rs/)
3.  **Windows Users only:** [C++ Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)
